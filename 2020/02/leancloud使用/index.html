LeanCloud使用

  2020-02-07   前端   服务器 数据存储  评论 字数统计: 723字 阅读时长: 2分

LeanCloud使用快速入门

简介

LeanCloud可以用来做一些简单的数据存储服务器，例如给页面添加留言等，主要用于存储文字这样的简单数据。

官方使用指南

LeanCloud官方数据存储开发指南 · JavaScript

快捷使用

安装

方法1、npm安装

# 存储服务（包括推送）官方源
$ npm install leancloud-storage --save
# 存储服务（包括推送）淘宝源
$ npm install leancloud-storage --save --registry=https://registry.npm.taobao.org

方法2、CDN引入

<!-- 存储服务 -->
<script src="//cdn.jsdelivr.net/npm/leancloud-storage@4.2.0/dist/av-min.js"></script>

初始化

首先进入 控制台 > 设置 > 应用 Key 来获取 App ID，App Key 以及服务器地址。

如果是在前端项目里面直接通过 CDN 加载的 LeanCloud JavaScript SDK，那么可以在页面加载的时候调用初始化的函数 AV.init：

AV.init({
  appId: "XXXXXXXXXXXXX",
  appKey: "XXXXXXXXXXXXX",
  serverURLs: "https://xxx.example.com"
});

测试

var TestObject = AV.Object.extend('TestObject');
var testObject = new TestObject();
testObject.set('words', 'Hello world!');
testObject.save().then(function (testObject) {
  console.log('保存成功。')
})

保存后运行程序。

然后打开 控制台 > 存储 > 数据 > TestObject，如果看到如下内容，说明 SDK 已经正确地执行了上述代码，安装完毕。

保存对象

下面的代码将一个 class 为 Todo 的对象存入云端：

// 声明 class
var Todo = AV.Object.extend('Todo');

// 构建对象
var todo = new Todo();

// 为属性赋值
todo.set('title', '马拉松报名');
todo.set('priority', 2);

// 将对象保存到云端
todo.save().then(function (todo) {
  // 成功保存之后，执行其他逻辑
  console.log('保存成功。objectId：' + todo.id);
}, function (error) {
  // 异常处理
});

为了确认对象已经保存成功，我们可以到 控制台 > 存储 > 数据 > Todo 里面看一下，应该会有一行新的数据产生。

获取对象

对于已经保存到云端的 AV.Object，可以通过它的 objectId 将其取回：

var query = new AV.Query('Todo');
query.get('582570f38ac247004f39c24b').then(function (todo) {
  // todo 就是 objectId 为 582570f38ac247004f39c24b 的 Todo 实例
  var title     = todo.get('title');
  var priority  = todo.get('priority');

  // 获取内置属性
  var objectId  = todo.id;
  var updatedAt = todo.updatedAt;
  var createdAt = todo.createdAt;
});

对象拿到之后，可以通过 get 方法来获取各个属性的值。注意 objectId、updatedAt和createdAt 这三个内置属性不能通过 get 获取或通过 set 修改，只能由云端自动进行填充。尚未保存的 AV.Object 不存在这些属性。
